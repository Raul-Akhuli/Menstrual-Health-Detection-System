# 🌸 Menstrual Health Detection System

An AI-powered web application that analyzes menstrual blood images to assist in early detection of potential health conditions using Deep Learning.

---

## 🌐 Live Demo

🚀 **Access the deployed application here:**  
https://blood-project-rosy.vercel.app/

---

## 📌 Overview

The Menstrual Health Detection System leverages Computer Vision and Machine Learning to provide early insights into menstrual health. Users can upload images and receive predictions about possible health conditions.

This project aims to:
- Improve awareness of menstrual health
- Provide early risk indicators
- Support preventive healthcare

---

## 🚀 Features

- 🧠 CNN-based image classification  
- 📷 Image preprocessing & normalization  
- 📊 Health condition prediction  
- 🌐 Fully deployed web application (Vercel)  
- ⚡ Fast and user-friendly interface  

---

## 🏗️ System Workflow

1. User uploads menstrual blood image  
2. Image preprocessing (resizing, normalization)  
3. Feature extraction using CNN  
4. Model prediction  
5. Display of health insights

## ⚙️ Installation & Setup (Local)

### 1. Clone the Repository

git clone https://github.com/Raul-Akhuli/Menstrual-Health-Detection-System.git  
cd Menstrual-Health-Detection-System  

### 2. Install Dependencies

pip install -r requirements.txt  

### 3. Run the Application

python app.py  

### 4. Open in Browser

http://localhost:5000

## 🧠 Model Details

- Model: Convolutional Neural Network (CNN)  
- Input: Menstrual blood images  
- Output: Predicted health condition  
- Training: Supervised Learning
